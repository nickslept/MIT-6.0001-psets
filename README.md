# MIT-6.0001-psets
My solutions to MIT OCW's 6.0001 psets. The course can be found [here](https://ocw.mit.edu/courses/6-0001-introduction-to-computer-science-and-programming-in-python-fall-2016/pages/syllabus/).

My solutions may not always be formatted to match up exactly with the responses they are looking for (e.g. printing out "months:" instead of "Time in months:", if they are seeking the latter). The code should (hopefully) still provide an acceptable solution to the problem.
